# 📑 目錄頁面範本設計教學

## 🎯 目標效果

創建一個像這樣的目錄頁面：

```
                        目錄

壹、企劃書之可行性及完整性
    一、專案緣起.............................................1-1
    二、專案願景.............................................1-1
    三、主要工作項目.........................................1-1
    四、對未來委辦事項規劃及設計構想...........................1-1
    五、執行方式及工作內容...................................1-54
    六、創意加值.............................................1-72

貳、資訊安全
    一、履約程序及環境之資安管理規劃及執行方式.................2-1
    二、履約相關之資安事件通報、應變、處理之規劃機制.............2-2
    三、SSDLC 監控與執行......................................2-6
    四、資安作為自評情形.....................................2-16

參、專案管理規劃
```

---

## 📝 第一步：在 Word 中創建基本結構

### 1. 開啟 Word，輸入以下內容：

```
目錄

{#chapters}
{title}
{#sections}
    {title}	頁碼待定
{/sections}

{/chapters}
```

---

## 🎨 第二步：設定格式

### A. 標題「目錄」

1. **選中「目錄」文字**
2. 設定格式：
   - 字體：**標楷體** 或 **微軟正黑體**
   - 大小：**18pt** 或 **20pt**
   - 粗體：**✓**
   - 對齊：**置中**
   - 下方間距：**12pt**（約一行）

### B. 章節標題（壹、貳、參...）

1. **選中 `{#chapters}` 下面的第一個 `{title}`**
2. 設定格式：
   - 字體：**標楷體** 或 **微軟正黑體**
   - 大小：**14pt**
   - 粗體：**✓**
   - 顏色：**黑色**
   - 段落間距：
     - 段落前：**12pt**
     - 段落後：**6pt**

### C. 子章節標題（一、二、三...）+ 點線 + 頁碼

這是**最重要**的部分！

#### 步驟 1：選中子章節那一行
```
    {title}	頁碼待定
```

#### 步驟 2：設定基本格式
- 字體：**標楷體** 或 **微軟正黑體**
- 大小：**12pt**
- 粗體：**否**
- 顏色：**黑色**

#### 步驟 3：設定縮排
1. 選中這一行
2. 點擊「常用」→「段落」群組右下角的小箭頭（開啟段落對話框）
3. 在「縮排」區域：
   - **左側縮排**：輸入 **1 字元** 或 **0.5 公分**
4. 點擊「確定」

**或者直接在文字前加 4 個空格**

#### 步驟 4：設定定位點（產生點線）

🔑 **這是關鍵步驟！**

1. 確保游標在子章節這一行
2. **方法 A：使用尺規**
   - 確保尺規已顯示（檢視 → 尺規 ✓）
   - 在尺規上**點擊左上角的定位點按鈕**，切換到「右對齊定位點」圖示
   - 在尺規的右側（例如 17 公分處）點擊一下，會出現一個定位點標記
   - **雙擊**這個定位點標記
   - 在「定位點」對話框中：
     - 對齊方式：**右對齊** ✓
     - 前置字元：選擇 **點線** ✓（通常是第 2 個選項）
   - 點擊「確定」

2. **方法 B：使用對話框**
   - 點擊「常用」→「段落」群組右下角的小箭頭
   - 點擊左下角的「定位點」按鈕
   - 或直接按 **Ctrl+T**（某些版本）
   - 在「定位點」對話框：
     - **定位點位置**：輸入 **17** 或 **16.5**（公分，根據你的頁面寬度調整）
     - **對齊方式**：選擇 **右對齊**
     - **前置字元**：選擇 **點線**（通常是第 2 個選項，顯示為 ......）
     - 點擊「設定」
     - 點擊「確定」

#### 步驟 5：插入 Tab 字元

**非常重要！**

1. 將游標放在 `{title}` 和 「頁碼待定」之間
2. 按下 **Tab 鍵**（鍵盤上的 ↹ 鍵）
3. 你應該會看到點線出現在標題和頁碼之間

**最終效果：**
```
    {title}................................頁碼待定
```

## 🔍 第三步：檢查範本

### 確認清單：

- [ ] 標題「目錄」已置中且格式正確
- [ ] 章節標題 `{title}` 格式正確（14pt、粗體）
- [ ] 子章節標題 `{title}` 有縮排（前面有空格或左縮排 0.5cm）
- [ ] 子章節標題和頁碼之間有 **Tab 字元**（不是空格）
- [ ] 有設定「右對齊定位點」且「前置字元」為「點線」
- [ ] 循環標記配對正確：`{#chapters}...{/chapters}` 和 `{#sections}...{/sections}`

---

## 💾 第四步：儲存範本

1. 檔案 → 另存為
2. 選擇位置：桌面
3. 檔案名稱：**目錄範本**
4. 儲存類型：**Word 文檔 (*.docx)**
5. 點擊「儲存」

---

## 📤 第五步：上傳到系統

1. 前往 `http://localhost:3000/dashboard/templates`
2. **雙擊**「ALL_RESOURCES」資料夾卡片
3. 填寫資訊：
   - 範本名稱：**目錄範本**
   - 分類：**目錄** 或 **文書格式**
   - 描述：**帶點線和頁碼的目錄頁面**
   - Word 檔案：選擇剛才儲存的「目錄範本.docx」
4. 點擊「確認上傳」
5. 在 OnlyOffice 編輯器中按 **Ctrl+S** 保存

---

## 🧪 第六步：測試範本

1. 前往 Planning 頁面
2. 在範本下拉選單選擇「目錄範本」
3. 點擊「預覽」按鈕
4. 檢查結果：
   - ✅ 章節標題顯示正確（壹、貳、參...）
   - ✅ 子章節有縮排（一、二、三... 往右縮）
   - ✅ 子章節標題和頁碼之間有**點線**（....）
   - ✅ 頁碼顯示在右側

## 🎨 進階：美化目錄

### 1. 添加裝飾線

在「目錄」標題下方和整個目錄下方添加分隔線：

```
═══════════════════════════════════
目錄
═══════════════════════════════════

{#chapters}
...
{/chapters}

─────────────────────────────────────
```

### 2. 調整間距

為了讓目錄更美觀：
- 章節之間間距：12pt
- 子章節之間間距：6pt
- 使用「段落」→「行與段落間距」設定

### 3. 添加頁碼說明

在目錄底部添加：
```
註：頁碼格式為「章節-頁數」，例如 1-1 表示第一章第一頁
```

## ⚠️ 常見問題排除

### Q1：點線沒有出現
**原因**：
- 沒有使用 Tab 字元（用了空格）
- 沒有設定定位點
- 定位點設定錯誤

**解決方法**：
1. 刪除 `{title}` 和「頁碼待定」之間的所有空格
2. 重新按 Tab 鍵
3. 確認已設定「右對齊定位點」且「前置字元」為「點線」

### Q2：子章節沒有縮排
**原因**：
- 沒有設定左側縮排
- 前面的空格被刪除了

**解決方法**：
1. 選中子章節的 `{title}` 行
2. 使用段落設定 → 左側縮排 → 0.5 公分
3. 或在 `{title}` 前面加 4 個空格

### Q3：定位點對話框找不到「前置字元」選項
**解決方法**：
1. 確保已經輸入了「定位點位置」（例如 17）
2. 先點擊「設定」按鈕，「前置字元」選項才會啟用
3. 如果還是找不到，嘗試：
   - 格式 → 段落 → 定位點
   - 或直接在尺規上雙擊定位點標記

### Q4：頁碼沒有對齊
**原因**：
- 定位點位置設定不正確

**解決方法**：
1. 根據你的頁面寬度調整定位點位置
   - A4 紙張，邊界 2.5cm → 定位點約 16cm
   - A4 紙張，邊界 2cm → 定位點約 17cm
2. 重新設定定位點

### Q5：填充後格式跑掉了
**原因**：
- 範本中的格式沒有正確套用到標記上

**解決方法**：
1. 確保選中 `{title}` **標記本身**時設定格式
2. 不要選中循環標記 `{#chapters}` 等
3. 格式只需設定在實際會被替換的變數上

## 📋 完整範本範例

```
═══════════════════════════════════
                目錄
═══════════════════════════════════

{#chapters}
{title}
{#sections}
    {title}	頁碼待定
{/sections}

{/chapters}

─────────────────────────────────────
註：頁碼格式為「章節-頁數」
═══════════════════════════════════
```

### 格式設定總結表

| 元素 | 字體 | 大小 | 粗體 | 顏色 | 對齊 | 縮排 | 其他 |
|------|------|------|------|------|------|------|------|
| 標題「目錄」 | 標楷體 | 18pt | ✓ | 黑色 | 置中 | - | - |
| 章節標題 | 標楷體 | 14pt | ✓ | 黑色 | 左 | - | 段落前 12pt |
| 子章節標題 | 標楷體 | 12pt | ✗ | 黑色 | 左 | 0.5cm | 右對齊定位點 17cm，點線 |
| 分隔線 | 任意 | 10pt | - | 灰色 | 置中 | - | - |

在練習 1 的基礎上，為章節標題添加點線和頁碼。

### 練習 3：完整目錄
完成上面的完整範本，包含章節、子章節、點線、頁碼。

## 💡 專業建議

### 1. 一致性
確保所有子章節的縮排、字體大小、點線樣式都完全一致。

### 2. 可讀性
- 不要讓內容過於密集
- 適當的行間距很重要（建議 1.15 倍或 1.5 倍）

### 3. 視覺層次
- 使用不同的字體大小和粗細區分層級
- 章節標題要明顯大於子章節標題

### 4. 測試
- 用不同長度的標題測試（短標題、長標題）
- 確保點線在各種情況下都正常顯示
