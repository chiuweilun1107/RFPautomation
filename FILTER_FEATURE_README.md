## ✨ 快速參考卡

### 狀態篩選
| 選項 | 顏色 | 用途 |
|------|------|------|
| 全部 | 黑色 | 所有標案 |
| 招標中 | 綠色 | 可投標案 |
| 已截止 | 紅色 | 歷史標案 |
| 已決標 | 藍色 | 決標結果 |

### 截止時間篩選
| 選項 | 顏色 | 優先級 |
|------|------|--------|
| 全部 | 黑色 | - |
| 3天內 | 紅色 | 最高 |
| 7天內 | 橙色 | 高 |
| 30天內 | 黃色 | 中 |

### 常用組合
- **今日緊急**：招標中 + 3天內
- **本週計畫**：招標中 + 7天內
- **本月總覽**：招標中 + 30天內
