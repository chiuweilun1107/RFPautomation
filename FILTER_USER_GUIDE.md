# 標案篩選功能使用指南

## 🎯 功能簡介

標案篩選功能幫助您快速找到最重要、最緊急的標案，提高投標效率。

### 主要功能
1. **狀態篩選** - 快速查看招標中、已截止或已決標的標案
2. **截止時間提醒** - 找到即將截止的緊急標案

---

## 📍 功能位置

打開標案列表頁面，在標題下方會看到兩排篩選按鈕：

```
┌─────────────────────────────────────────────────────┐
│ STATUS_FILTER:  [全部] [招標中] [已截止] [已決標]   │
│ DEADLINE_ALERT: [全部] [3天內] [7天內] [30天內]     │
└─────────────────────────────────────────────────────┘
```

---

## 🔘 狀態篩選使用說明

### 全部（預設）
- **作用**：顯示所有標案，不論狀態
- **使用時機**：想要瀏覽所有可用標案時
- **按鈕顏色**：黑色

### 招標中
- **作用**：僅顯示尚未截止的標案
- **篩選條件**：截止日期在未來，或無截止日期
- **使用時機**：想要找到還能投標的標案
- **按鈕顏色**：綠色 - 表示「安全可投」
- **範例**：今天是 1/28，會顯示截止日期在 1/29 以後的標案

### 已截止
- **作用**：僅顯示已經過了截止日期的標案
- **篩選條件**：截止日期在過去
- **使用時機**：檢視歷史標案、分析市場趨勢
- **按鈕顏色**：紅色 - 表示「已逾期」
- **範例**：今天是 1/28，會顯示截止日期在 1/27 以前的標案

### 已決標
- **作用**：僅顯示已完成決標的標案
- **篩選條件**：狀態欄位包含「已決標」
- **使用時機**：查看決標結果、了解競爭對手
- **按鈕顏色**：藍色 - 表示「已完成」

---

## ⏰ 截止時間提醒使用說明

### 全部（預設）
- **作用**：不進行時間篩選
- **使用時機**：想要看到所有時間範圍的標案
- **按鈕顏色**：黑色

### 3天內截止
- **作用**：僅顯示未來3天內截止的標案
- **篩選條件**：今天到截止日期 ≤ 3天
- **使用時機**：查看最緊急、需立即處理的標案
- **按鈕顏色**：紅色 - 最高優先級
- **範例**：今天是 1/28，會顯示 1/28-1/31 截止的標案
- **提醒**：這些標案需要立即準備投標文件！

### 7天內截止
- **作用**：僅顯示未來7天內截止的標案
- **篩選條件**：今天到截止日期 ≤ 7天
- **使用時機**：規劃本週投標計畫
- **按鈕顏色**：橙色 - 中高優先級
- **範例**：今天是 1/28，會顯示 1/28-2/4 截止的標案

### 30天內截止
- **作用**：僅顯示未來30天內截止的標案
- **篩選條件**：今天到截止日期 ≤ 30天
- **使用時機**：規劃本月投標策略、預估工作量
- **按鈕顏色**：黃色 - 中等優先級
- **範例**：今天是 1/28，會顯示 1/28-2/27 截止的標案

## 🔗 篩選組合使用技巧

### 技巧 1：找到最緊急的標案
```
操作：「招標中」+ 「3天內」
結果：僅顯示還能投標且3天內截止的標案
適用：每天早上檢查待辦事項
```

**實際範例**：
- 今天是 1/28
- 標案 A：截止 1/30，狀態「招標中」→ ✓ 顯示（緊急！）
- 標案 B：截止 1/30，狀態「已決標」→ ✗ 不顯示（已結束）
- 標案 C：截止 2/5，狀態「招標中」→ ✗ 不顯示（超過3天）

### 技巧 2：規劃本週工作
```
操作：「招標中」+ 「7天內」
結果：本週需要準備的所有標案
適用：每週一早上規劃本週投標計畫
```

### 技巧 3：本月投標策略
```
操作：「招標中」+ 「30天內」+ 切換到「行事曆視圖」
結果：行事曆上顯示本月所有可投標案的截止日期
適用：每月初規劃本月投標節奏
```

### 技巧 4：市場分析
```
操作：「已決標」
結果：已完成決標的所有標案
適用：分析競爭對手得標情況、市場趨勢
```

### 技巧 5：特定關鍵字的緊急標案
```
操作：
1. 選擇關鍵字（例如「軟體開發」）
2. 點擊「招標中」
3. 點擊「3天內」
結果：軟體開發類別中3天內截止的標案
```

## 💡 篩選狀態提示列

當您啟用任何篩選時，頁面上方會顯示紅色提示列：

```
┌──────────────────────────────────────────────────────────────┐
│ Active_Filters: STATUS: 招標中 DEADLINE: 3天內              │
│ // 8 TENDERS_FOUND                        [Clear_All [X]]   │
└──────────────────────────────────────────────────────────────┘
```

**提示列內容**：
- **Active_Filters**：顯示目前啟用的篩選
- **數量**：顯示符合條件的標案數量
- **Clear All 按鈕**：一鍵清除所有篩選

## ⚠️ 注意事項

### 關於截止時間篩選
- **只顯示未來的日期**：截止時間篩選不會顯示已過期的標案
- **範例**：選擇「3天內」時，已經過了截止日期的標案不會顯示

### 關於無截止日期的標案
- **「招標中」**：會顯示無截止日期的標案（視為進行中）
- **「已截止」**：不會顯示無截止日期的標案
- **截止時間篩選**：不會顯示無截止日期的標案

### 篩選與搜尋的關係
- 篩選與搜尋**同時生效**（AND 邏輯）
- 範例：搜尋「維護」+ 「招標中」= 標題包含「維護」且招標中的標案

### 篩選與關鍵字的關係
- 先選擇關鍵字，再應用篩選
- 篩選只作用於該關鍵字的標案

## 🔄 清除篩選

### 方法 1：點擊「全部」
- 點擊狀態篩選的「全部」
- 點擊截止時間提醒的「全部」

### 方法 2：使用「Clear All」按鈕
- 點擊提示列右側的「Clear_All_Filters [X]」按鈕
- 一次清除所有篩選條件

## ❓ 常見問題

### Q1：為什麼「3天內」沒有顯示任何標案？
**A1**：可能的原因：
- 目前沒有3天內截止的標案
- 您選擇的關鍵字沒有緊急標案
- 嘗試選擇「7天內」或「30天內」查看

### Q2：「招標中」和「已截止」有重疊嗎？
**A2**：不會。這兩個選項是互斥的：
- 「招標中」：截止日期 > 現在
- 「已截止」：截止日期 ≤ 現在

### Q3：可以同時選擇多個狀態篩選嗎？
**A3**：不可以。狀態篩選是單選的，但可以搭配截止時間篩選使用。

### Q4：篩選會影響分頁嗎？
**A4**：會。分頁是根據篩選後的結果計算的。

### Q5：清除篩選後會回到第一頁嗎？
**A5**：會。清除篩選會自動重置到第一頁。

## 🚀 快速參考

### 顏色代碼
| 顏色 | 含義 | 使用場景 |
|------|------|----------|
| 綠色 | 安全可投 | 招標中 |
| 紅色 | 緊急/警告 | 已截止、3天內 |
| 藍色 | 資訊/完成 | 已決標 |
| 橙色 | 注意 | 7天內 |
| 黃色 | 提醒 | 30天內 |

### 優先級建議
1. **最高優先**：「招標中」+ 「3天內」（紅色）
2. **高優先**：「招標中」+ 「7天內」（橙色）
3. **中優先**：「招標中」+ 「30天內」（黃色）
4. **低優先**：「招標中」（綠色）

### 快捷鍵（建議實作）
- `Ctrl+1`：招標中
- `Ctrl+2`：已截止
- `Ctrl+3`：已決標
- `Ctrl+4`：3天內
- `Ctrl+0`：清除所有篩選
