# Unit Testing Commands Reference Card
# =====================================

## Running Tests
--------------
npm test                     # Run all tests once
npm run test:watch           # Watch mode (auto-rerun)
npm run test:coverage        # Generate coverage report
npm run test:ci              # CI/CD mode (for GitHub Actions)

## Specific Test Runs
------------------
npm test -- button           # Run tests matching "button"
npm test -- src/__tests__/lib/utils.test.ts  # Run specific file
npm test -- --verbose        # Verbose output
npm test -- -u               # Update snapshots

## Coverage
---------
npm run test:coverage                    # Generate report
open coverage/lcov-report/index.html     # View in browser (Mac)

## Test Development
----------------
npm run test:watch                       # Auto-rerun on file change
npm test -- --testNamePattern="should"   # Run specific test names
npm test -- --onlyChanged                # Only changed files

## Debug Mode
-----------
node --inspect-brk node_modules/.bin/jest --runInBand
# Then open chrome://inspect in Chrome

## Quick Status Check
------------------
npm test 2>&1 | grep "Test Suites"      # Summary only
npm test 2>&1 | grep -E "PASS|FAIL"     # Pass/fail status

## Test File Locations
-------------------
src/__tests__/components/     # Component tests
src/__tests__/hooks/          # Hook tests
src/__tests__/lib/            # Utility tests
