# 服務企劃書 - 帶目錄的範本設計

## 完整範本內容（複製到 Word）

```
═══════════════════════════════════
服務企劃書
═══════════════════════════════════


📋 目  錄
─────────────────────────────────

{#chapters}
{title} ............................ 第 {order_index} 章

  {#sections}
  　{title}

  {/sections}

{/chapters}

─────────────────────────────────


═══════════════════════════════════
詳細內容
═══════════════════════════════════


{#chapters}
【第 {order_index} 章】 {title}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  {#sections}
  ▸ {title}
  ─────────────────────────────

  {content}

  {/sections}

{/chapters}


═══════════════════════════════════
報告結束
═══════════════════════════════════
```

## 格式設定步驟

### 1. 標題部分
- **「服務企劃書」**
  - 字體：微軟正黑體
  - 大小：24pt
  - 粗體：是
  - 顏色：黑色
  - 對齊：置中

### 2. 目錄部分
- **「📋 目  錄」**
  - 字體：微軟正黑體
  - 大小：18pt
  - 粗體：是
  - 顏色：#FA4028 (紅色)

- **目錄中的 {title}**（第一個出現在目錄中的）
  - 字體：微軟正黑體
  - 大小：14pt
  - 粗體：是
  - 顏色：黑色

- **目錄中第二個 {title}**（小節標題）
  - 字體：微軟正黑體
  - 大小：12pt
  - 粗體：否
  - 顏色：#666666 (灰色)
  - 前面有兩個全形空格（　）

### 3. 內容部分
- **「【第 X 章】{title}」**（詳細內容中的章節標題）
  - 字體：微軟正黑體
  - 大小：20pt
  - 粗體：是
  - 顏色：#FA4028 (紅色)

- **「▸ {title}」**（小節標題）
  - 字體：微軟正黑體
  - 大小：16pt
  - 粗體：是
  - 顏色：黑色

- **{content}**
  - 字體：微軟正黑體
  - 大小：12pt
  - 粗體：否
  - 顏色：黑色
  - 行距：1.5

## 注意事項

⚠️ **重要：order_index 變數目前不支援**

目前系統只傳遞：
- chapters (陣列)
- title (標題)
- sections (陣列)
- content (內容)

如果你需要章節編號，請在標題中直接寫「壹、」、「貳、」等，不要使用 {order_index}。

簡化版本（建議使用）：

```
📋 目  錄
─────────────────────────────────

{#chapters}
{title}

  {#sections}
  　{title}

  {/sections}

{/chapters}
```
