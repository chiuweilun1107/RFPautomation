## 🎨 顏色語義化

### 狀態顏色
| 狀態   | 顏色代碼 | 含義           | 使用場景                |
|--------|----------|----------------|-------------------------|
| 黑色   | #000000  | 預設/中性      | 「全部」選項            |
| 綠色   | #00C853  | 安全/進行中    | 「招標中」標案          |
| 紅色   | #FA4028  | 警告/截止      | 「已截止」、緊急提醒    |
| 藍色   | #285AFA  | 資訊/完成      | 「已決標」              |
| 橙色   | #FF9800  | 注意/中等優先  | 「7天內截止」           |
| 黃色   | #FDD835  | 提醒/低優先    | 「30天內截止」          |

### 設計原則
1. **色彩一致性**：遵循 Swiss Design 的簡潔配色
2. **語義化**：顏色傳達明確的資訊層級
3. **對比度**：確保文字可讀性（WCAG AA 標準）
4. **狀態清晰**：選中狀態與未選中狀態有明顯區別

## 🔤 文字樣式

### 標籤文字（STATUS_FILTER:, DEADLINE_ALERT:）
```css
font-size: 10px;
font-family: monospace;
font-weight: 900; /* black */
text-transform: uppercase;
letter-spacing: 0.1em;
color: rgba(0, 0, 0, 0.4); /* 40% opacity */
```

### 按鈕文字
```css
font-size: 10px;
font-family: monospace;
font-weight: 900; /* black */
text-transform: uppercase;
letter-spacing: 0.05em;
```

### 提示列文字
```css
font-size: 10px (標籤) / 14px (值);
font-family: monospace;
font-weight: 900; /* black */
text-transform: uppercase;
color: white;
```

## 💡 交互狀態

### 按鈕狀態轉換
```
預設狀態（未選中）：
  背景：透明
  文字：灰色 (text-muted-foreground)
  邊框：繼承父容器

懸停狀態：
  背景：淺灰 (bg-muted)
  文字：不變
  過渡：transition-colors

選中狀態：
  背景：對應顏色（黑/綠/紅/藍/橙/黃）
  文字：白色 (或黑色 for 黃色)
  邊框：無
```

### 清除按鈕
```
預設：
  背景：透明
  邊框：白色 (border-white)
  文字：白色

懸停：
  背景：白色 (bg-white)
  文字：紅色 (text-[#FA4028])
```

## 📱 響應式斷點

### 桌面 (≥ 640px)
```
篩選面板：水平排列 (flex-row)
兩個篩選組並排顯示
```

### 平板/手機 (< 640px)
```
篩選面板：垂直堆疊 (flex-col)
狀態篩選在上
截止時間篩選在下
按鈕可能需要橫向滾動或換行
```

## 🎯 無障礙考量

### 鍵盤導航
- 所有按鈕支援 Tab 鍵導航
- Enter/Space 鍵啟用按鈕
- 使用 `<button>` 元素確保語義正確

### 螢幕閱讀器
- 標籤使用語義化文字（STATUS_FILTER, DEADLINE_ALERT）
- 按鈕文字清晰描述功能
- 提示列提供篩選結果資訊

### 顏色對比
- 所有文字與背景的對比度 ≥ 4.5:1
- 選中狀態與未選中狀態有明顯區別
- 不依賴顏色作為唯一的資訊來源

## 🔍 視覺層級

```
層級 1（最高）：提示列 - 紅色背景，吸引注意
層級 2：選中的篩選按鈕 - 有色背景
層級 3：篩選標籤 - 40% 透明度文字
層級 4：未選中的按鈕 - 灰色文字
```
